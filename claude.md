# massash.com — Project Brain

## Видение проекта

massash.com — платформа для специалистов в области массажа, реабилитации, тренерства и медицины.
Специалисты из разных городов России размещают профили, ведут клиентов, продвигаются внутри платформы.

Александр (владелец) — первый специалист на платформе. Этап 1 — его личный сайт.
Этап 2 — каталог специалистов. Этап 3 — CRM и личный кабинет.

---

## Три фазы развития

### Этап 1 — Личный сайт специалиста (сейчас)
- Стек: HTML + CSS + JS, статика, без бэкенда
- Страницы: главная, реабилитация, массаж, о специалисте
- Форма записи → n8n webhook → Telegram
- Деплой: AdminVPS (87.236.16.134), домен massash.com (DNS на Beget)

### Этап 2 — Каталог специалистов
- Стек: Python FastAPI + PostgreSQL + Jinja2
- Типы специалистов: массажисты, реабилитологи, тренеры, врачи
- Географии: разные города России
- Монетизация: платное размещение, внутреннее продвижение (модель base-massage.com)
- HTML Этапа 1 становится Jinja2-шаблонами без переписки

### Этап 3 — Платформа ведения клиентов
- Личный кабинет специалиста
- CRM: карточки клиентов, прогресс, программы реабилитации
- Модель: trnr.ru
- Бэкенд: Python (расширение Этапа 2)

## Дизайн-система

### Палитра
```css
--cream: #F5F0E8;       /* фон — тёплый кремовый */
--dark: #1C1915;        /* текст — почти чёрный */
--terra: #B5522A;       /* акцент — терракота */
--terra-light: #D4795A; /* акцент светлый */
--muted: #8A7E6F;       /* второстепенный текст */
--border: rgba(28,25,21,0.12);
--card-bg: #FFFDF8;
```

### Шрифты
- Заголовки: `Cormorant Garamond` (Google Fonts) — элегантность, классика
- Текст: `Nunito Sans` (Google Fonts) — читаемость, доверие

### Принципы
- Анимации медленные (0.3–0.5s), дыхательные
- Сетка: CSS Grid, без Bootstrap
- Мобиль: media query 768px, одна колонка

## Контент — реабилитация (rehabilitation.html)

### Три уровня сложности

**Уровень 1 — Биомеханическая реабилитация**
Работа с опорно-двигательным аппаратом без неврологического дефицита.
Инструменты: массаж, ПИР, ЛФК.
Программы: Осанка, Остеохондроз, Плечо-рука, Плоскостопие, Травмы конечностей

**Уровень 2 — Нейрореабилитация**
Последствия поражения нервной системы. Связка с врачом обязательна.
Программы: Инсульт, ЧМТ

**Уровень 3 — Постоперационная реабилитация**
Строго по протоколу хирурга. Высокая юридическая ответственность.
Программы: После операций на позвоночнике, Эндопротезирование

## Правила разработки

1. Не усложнять — Этап 1 статика, без фреймворков
2. Структура URL не меняется — заложена под Этапы 2 и 3
3. CSS через переменные — вся палитра в variables.css
4. После каждого изменения — git commit
5. Мобильная версия обязательна — media query 768px
6. Форма только через n8n webhook — не PHP, не сторонние сервисы
7. Не добавлять зависимости без необходимости
